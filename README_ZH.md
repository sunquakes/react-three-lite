## 安装

### 安装 `Three.js`

```bash
pnpm i three
```

### 安装 `react-three-lite`

```bash
pnpm i react-three-lite
```

## 快速开始

- 在 `main.js` 中从 `react-three-lite` 导入所需的组件。

```jsx
import { Scene } from 'react-three-lite'

function App() {
  return <Scene style={{ width: '100%', height: '300px' }} />
}
```
