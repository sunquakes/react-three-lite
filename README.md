## Install

### Install `Three.js`

```bash
pnpm i three
```

### Install `react-three-lite`

```bash
pnpm i react-three-lite
```

## Getting Started

- Import the desired components from `react-three-lite` in `main.js`.

```jsx
import { Scene } from 'react-three-lite'

function App() {
  return <Scene style={{ width: '100%', height: '300px' }} />
}
```
